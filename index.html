  • About the Project

    The flight route mapping project is an innovative approach to visualize the top 25 busiest airports in India. Using this project, one can easily view and analyze the routes and traffic of these airports in a dynamic and interactive way. The project leverages the power of data visualization to provide insights into the busy airports' air traffic and route patterns. The project has a comprehensive and intuitive interface, displaying the air routes in a map layout and highlighting the airports with a label for ease of interpretation.

    The project's data comes from CSV files that contain the starting and ending points of the air routes, which are then interpolated and plotted on the map. The project uses a scatter plot to display the starting and ending points, and the lines connecting the points to show the route taken by the planes.

Tech Stack

Python

We used Python to import, clean, and manipulate data from various sources and then used it to create visualizations of the data.Basemap is a library in Python that provides a way to create maps and plot geospatial data.

Pandas

Pandas is an open-source data manipulation library for Python that we used to read in and clean the CSV files containing the flight route data. We also used Pandas to organize and manipulate the data for use in the visualizations.

Matplotlib

Matplotlib is a plotting library for Python that we used to create static visualizations of the flight route data. We used Matplotlib to create a scatter plot of the airports and to plot the routes between the airports..

Plotly

Plotly is an open-source data visualization library that we used to create interactive visualizations of the flight route data. We used Plotly to create an interactive map of India with markers for the airports and lines to represent the routes between them.

Flourish Studio's Bar Chart Race

You used Flourish Studio's Bar Chart Race to create an animated visualization of the top 25 busiest airports in India over time. We imported the data into Flourish Studio and used their user-friendly interface to create the animation.

DaVinci Resolve

DaVinci Resolve is a video editing software that we used to edit the final video for our project. We imported the visualizations we created in Python and Flourish Studio and used DaVinci Resolve to combine and edit them into a cohesive video presentation..

The Methodology

The flight route mapping project is an innovative approach to visualize the top 25 busiest airports in India, displaying their air routes and traffic patterns in a dynamic and interactive way. The project uses a range of techniques and tools to achieve its goal, including Python, Pandas, Matplotlib, Plotly, Flourish Studio's Bar Chart Race, and DaVinci Resolve.

The methodology behind the project starts with data collection. Data on the air traffic routes was collected from a variety of sources, including airport databases and aviation industry reports. The data was compiled into a CSV file, which was then processed and analyzed using Python and its data analysis libraries, Pandas and Matplotlib.

The data was cleaned and organized to prepare it for visualization. The cleaned data was then loaded into Pandas DataFrames and analyzed using Matplotlib to create various charts and graphs. These charts provided an initial understanding of the data and helped to identify key trends and patterns.

Next, the data was visualized using Plotly, a Python library for creating interactive data visualizations. Plotly allowed the project team to create interactive maps that could be zoomed, panned, and rotated, enabling a more in-depth analysis of the data. The data was plotted on an interactive map, using Plotly's mapping capabilities, to provide a visual representation of the air routes and traffic patterns of the busiest airports in India.

To add an extra dimension to the visualization, Flourish Studio's Bar Chart Race was used to create an animated bar chart race, which allowed the team to showcase the changes in traffic over time. The chart race provided a dynamic and engaging way to present the data and allowed viewers to see the changing patterns of air traffic over time.

Finally, the project was completed with the creation of a video presentation that showcased the visualization and provided an overview of the findings. DaVinci Resolve, a professional video editing software, was used to create the final video.

DATA SOURCE

The data source used in the project is the flight schedule page of GoIndigo, one of India's leading low-cost airlines. The page contains information about IndiGo's flight schedules, including flight numbers, departure and arrival times, and the days on which the flights operate.

Future Work

Integration with real-time data: Currently, the project visualizes flight routes based on data from a specific time period. In the future, the project could be enhanced to integrate with real-time flight data so that users can see the current flight routes and traffic patterns.

Expansion to other countries/regions: While the project currently focuses on the top 25 busiest airports in India, it could be expanded to cover other countries and regions. This would provide a more comprehensive view of air traffic patterns and could be useful for analyzing trends and making predictions.

Addition of more interactive features: The project could be enhanced with additional interactive features such as the ability to filter data based on specific criteria, or to view more detailed information on specific flights or airports.

Optimization for mobile devices: While the project is currently optimized for desktop viewing, it could be further optimized for mobile devices to ensure a seamless user experience across all devices.

Integration with other data sources: The project could be enhanced by integrating with other data sources such as weather data or flight delay information. This could provide a more comprehensive view of the factors that impact air traffic patterns.
